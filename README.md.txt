## 1. Project Purpose
Analyze and compare the financial performance of Midea, Gree, Haier using key ratios:
- Net Profit Margin
- ROE
- Asset Turnover

Target users: Finance students, investors, analysts.

## 2. Data Source
- Database: WRDS (wrdsapps.sa0)
- Access date: 2026-04-20
- Period: 2018–2024
- Companies: Haier(600690), Gree(000651), Midea(000333)

## 3. Python Workflow
1. Connect to WRDS

3. Data cleaning
4. Calculate financial ratios
5. Visualization
6. Export to Excel

## 4. Key Findings

- Midea has the best asset turnover efficiency.
